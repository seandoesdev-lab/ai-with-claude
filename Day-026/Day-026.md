## 0. 들어가며 (Motivation)

지난 편([[Day-025]])에서 Phase 2 를 닫으며 우리는 한 문장을 남겼습니다: **"점수의 근거를 *글자 교집합* 에서 *학습된 의미 벡터의 가까움* 으로 바꿔야 한다."** 그 "학습된 의미 벡터" 를 만들어 내는 기계가 바로 **신경망(neural network)** 입니다. 그런데 신경망이라 하면 트랜스포머·GPT 같은 거대한 것부터 떠올리기 쉽지만, 그 모두는 **아주 단순한 부품 하나** — 뉴런(neuron) — 를 수백만 개 쌓아 만든 것입니다.

좋은 소식은, 그 부품이 **우리가 이미 아는 것** 이라는 점입니다. [[Day-004]] 에서 만난 내적 $w^\top x$, [[Day-014]] 에서 학습시킨 로지스틱 회귀의 $w^\top x + b$ — **뉴런 하나가 정확히 이 식** 입니다. 즉 Phase 1~2 에서 손에 익힌 선형 모델이 신경망의 "한 층(one layer)" 으로 자연스럽게 확장됩니다. 오늘은 그 확장의 **첫 계단** 을 밟습니다: 뉴런 하나 → 층 → 층을 쌓기 → **왜 비선형이 있어야 쌓는 의미가 생기나**.

> Phase 3 starts here. A neural network is millions of copies of one tiny part — a neuron — and that part is something we already know: the linear score $w^\top x + b$ from [[Day-004]] and [[Day-014]]. Today we go from one neuron to a multi-layer network, using the XOR problem to see *why* we must stack layers **and** why a nonlinearity is non-negotiable.

## 1. 핵심 개념 (Core Concepts)

### 1.1 뉴런 하나 = 선형 모델 + 활성함수

인공 뉴런(artificial neuron)이 하는 일은 딱 두 단계입니다:

$$
z \;=\; w^\top x + b \;=\; \sum_{i} w_i x_i + b
\qquad\Longrightarrow\qquad
a \;=\; \phi(z)
$$

1. **가중합(weighted sum)** $z = w^\top x + b$ — 입력 $x$ 에 가중치 $w$ 를 곱해 더하고 편향 $b$ 를 얹습니다. **이 식은 [[Day-004]] 의 내적, [[Day-014]] 의 로지스틱 회귀 점수와 글자 그대로 같습니다.**
2. **활성함수(activation function)** $a = \phi(z)$ — 가중합을 비선형 함수 $\phi$ 에 통과시킵니다. 초창기 퍼셉트론은 계단 함수(step), 오늘날은 주로 **ReLU** $\phi(z)=\max(0,z)$ 를 씁니다.

$w$ 와 $b$ 를 **파라미터(parameter)** 라 부르며, "학습" 이란 곧 데이터를 보고 이 $w, b$ 를 조정하는 일입니다(그 방법이 다음 편의 역전파).

### 1.2 퍼셉트론과 '선형 분리 가능성'

뉴런 하나에 계단 활성함수를 붙인 것이 **퍼셉트론(perceptron, Rosenblatt 1958)** 입니다. 결정 경계는 $w^\top x + b = 0$ 이라는 **직선(초평면, hyperplane)** 하나입니다. 그래서 퍼셉트론은 **직선 하나로 두 부류를 가를 수 있는(선형 분리 가능, linearly separable)** 문제만 풉니다.

| 문제 | 직선 하나로 가능? | 뉴런 하나로 |
|------|------------------|------------|
| AND, OR | ✅ 가능 | 풀림 |
| **XOR** | ❌ 불가능 | **못 풂** |

이 한계는 1969년 민스키·페이퍼트의 *Perceptrons* 가 지적하며 첫 "AI 겨울" 의 빌미가 되었고, 해법(=은닉층+비선형+역전파)이 나온 뒤에야 딥러닝이 부활합니다.

### 1.3 XOR — 뉴런 하나가 무너지는 지점

XOR(배타적 논리합)의 정답은 $(0,0)\to0,\ (0,1)\to1,\ (1,0)\to1,\ (1,1)\to0$ 입니다. 정답이 1인 점 $(0,1),(1,0)$ 은 **대각선으로 마주보고**, 0인 점 $(0,0),(1,1)$ 도 대각선으로 마주봅니다. **어떤 직선을 그어도** 1들만 한쪽에 모을 수 없습니다 — 선형 분리 불가능. 그래서 $w,b$ 를 어떻게 골라도 뉴런 하나는 4개 중 최대 3개까지만 맞힙니다(실습 (2)에서 확인).

### 1.4 층을 쌓기 — 그런데 그냥 쌓으면 소용없다 (선형 붕괴)

"뉴런 하나가 안 되면 여러 개를 여러 층으로 쌓자" — 맞는 방향입니다. 그런데 **활성함수 없이** 선형 층만 쌓으면 함정에 빠집니다:

$$
\text{2층: } \; W_2\,(W_1 x) \;=\; (W_2 W_1)\, x \;=\; W\, x
$$

행렬 곱은 결합법칙이 성립하므로, 선형 층을 아무리 많이 쌓아도 **하나의 선형 층 $W = W_2 W_1$ 로 붕괴(collapse)** 합니다. 표현력은 그대로 선형 — XOR 는 여전히 못 풉니다. (실습 (3)에서 $W_2(W_1x)=(W_2W_1)x$ 를 수치로 확인합니다.)

### 1.5 비선형성이 열쇠 — MLP

층 사이에 **비선형 활성함수** $\phi$ 를 끼우면 붕괴가 깨집니다:

$$
h = \phi(W_1 x + b_1), \qquad y = W_2 h + b_2
$$

$\phi$ 때문에 $W_2\phi(W_1x)$ 는 더 이상 하나의 선형식으로 합쳐지지 않습니다. 이렇게 **입력층 → 은닉층(hidden layer) → 출력층** 으로 이어진 구조가 **다층 퍼셉트론(MLP, Multi-Layer Perceptron)** 이고, 은닉층 덕분에 **곡선(비선형) 결정 경계** 를 그릴 수 있어 XOR 도 풀립니다. 실제로 ReLU 은닉 뉴런 2개면 충분합니다(실습 (4)).

> **보편 근사 정리(Universal Approximation Theorem):** 은닉층 하나에 비선형 활성함수만 있으면, 뉴런 수를 늘려 어떤 연속 함수든 원하는 정밀도로 근사할 수 있다. 즉 "비선형 + 은닉층" 이 표현력의 원천이다.

### 1.6 순전파(Forward Pass)

입력을 넣어 층을 **앞으로(front-to-back)** 통과시키며 출력을 계산하는 과정이 **순전파(forward pass)** 입니다. 오늘 다루는 건 순전파까지 — 즉 *가중치가 주어졌을 때 답을 내는* 부분입니다. 그 가중치를 *데이터로 스스로 찾게* 하는 반대 방향 과정이 **역전파(backpropagation)** 이며, 바로 다음 편([[Day-027]])의 주제입니다.

## 2. 직관 / 비유 (Intuition)

> [!tip] 비유 — 종이 한 장 접기로 점 가르기
> 평평한 종이 위에 점 4개(XOR)가 있습니다. **가위질 한 번(직선 하나=뉴런 하나)** 으로는 1인 두 점만 따로 오려낼 수 없습니다 — 대각선으로 흩어져 있으니까요. 그런데 **종이를 한 번 접으면(비선형 변환=은닉층+ReLU)** 멀리 있던 두 점이 포개져 이웃이 되고, 그제서야 **가위질 한 번(출력층의 직선)** 으로 깔끔히 갈립니다. 은닉층이 하는 일이 바로 이 "접기" — 입력 공간을 비틀어, 원래 선형으로 못 가르던 것을 가를 수 있는 새 공간으로 옮기는 것입니다. 활성함수(비선형)가 없으면 종이를 접지 못하고 밀기만 하니(선형 변환), 아무리 여러 번 해도 평평한 채라 소용이 없습니다.
>
> One straight cut (a single neuron) can't separate XOR's diagonal points. But *folding* the paper first (a hidden layer with a nonlinearity) brings them together so one final cut works. The hidden layer folds the input space; without a nonlinearity you can only slide the paper (a linear map), never fold it — which is why stacking linear layers collapses to one.

## 3. 한 장의 그림 (One Picture)

```text
  입력 x = (x1, x2)
     │  ┌───────────────── 뉴런 하나 = 선형 + 활성 ─────────────────┐
     │  │   z = w·x + b   →   a = φ(z)     (Day-004/Day-014 의 w·x+b) │
     │  └────────────────────────────────────────────────────────────┘
     │            └ 직선 하나로만 가름 → AND ✅ , XOR ❌
     ▼
  ┌──────────── MLP (은닉층 + 비선형) ─────────────┐
  │   은닉:  h = ReLU(W1·x + b1)   ← '종이 접기'(비선형)│
  │   출력:  y =      W2·h  + b2                        │
  └──────────────────────────────────────────────────────┘
        │  비선형 φ 가 없으면:  W2(W1x) = (W2W1)x = Wx  (선형 붕괴!)
        ▼
   XOR 도 풀린다 → Phase 3 계속: 역전파 → RNN → Attention → Transformer

   순전파(forward): 입력 ──▶ 은닉 ──▶ 출력   (오늘)
   역전파(backprop): 오차 ◀── 기울기 전파 ◀── 출력  (다음 편, Day-027)
```

> A neuron is just $w^\top x+b$ followed by an activation — it draws one straight boundary, enough for AND but not XOR. An MLP inserts a hidden layer with a **nonlinearity** (the "paper fold"); without it, stacked linear layers collapse to $Wx$. Forward pass runs input→hidden→output; the reverse (backprop) is next.

### 4.2 핵심 코드 — 뉴런 한 개와 MLP 순전파 (발췌)

```python
import numpy as np

def relu(z):                      # 비선형 활성함수 (딥러닝의 심장)
    return np.maximum(0.0, z)

# --- 뉴런 하나 = 선형 모델 w·x + b + 활성함수 ---
def neuron(x, w, b, act):
    z = float(np.dot(w, x)) + b   # w^T x + b  (Day-004/Day-014 와 같은 식!)
    return act(np.array(z))

# --- ReLU 를 끼운 2층 MLP 로 XOR 풀기 (순전파) ---
W1 = np.array([[1., 1.], [1., 1.]]); b1 = np.array([0., -1.])   # 은닉층
W2 = np.array([[1., -2.]]);          b2 = np.array([0.])        # 출력층
for x1, x2 in [(0,0),(0,1),(1,0),(1,1)]:
    x = np.array([x1, x2], float)
    h = relu(W1 @ x + b1)         # 은닉층 순전파 (비선형!)
    y = float((W2 @ h + b2)[0])   # 출력층 순전파 (선형)
    print(x1, x2, "->", round(y))  # XOR: 0,1,1,0
```

```
------------------------------------------------------------------------

------------------------------------------------------------------------

  W2 @ (W1 @ x) = [26.0, 6.5]   (2개 층을 차례로)
  (W2 @ W1) @ x = [26.0, 6.5]   (하나로 합친 층)
  두 결과가 같은가? -> True

  은닉:  h = ReLU(W1·x + b1),   W1=[[1,1],[1,1]], b1=[0,-1]

        출력에서 -2 로 눌러 (1,1)을 0 으로 되돌린다.

========================================================================
결론: 뉴런 하나 = 선형 모델(w·x+b) + 활성함수.  선형 문제(AND)는 풀지만
      XOR 같은 비선형 문제는 못 푼다.  층을 쌓되 '비선형 활성함수'를 끼워야
      비로소 표현력이 커진다(선형 붕괴 회피).  이것이 MLP·딥러닝의 출발점.
      다음: 이 가중치를 '데이터로 학습'하는 법 = 역전파(backpropagation).
========================================================================
```

### 4.3 결과를 정직하게 읽기 (관찰 포인트)

1. **(1) 뉴런 하나로 충분한 경우.** AND 는 직선 $x_1+x_2=1.5$ 하나로 $(1,1)$ 만 떼어내면 되므로 뉴런 하나로 완벽합니다. 여기서 $z=w^\top x+b$ 는 [[Day-014]] 의 로지스틱 회귀 점수와 **똑같은 식** — 신경망은 우리가 아는 선형 모델에서 시작합니다.
2. **(2) 뉴런 하나가 무너지는 순간.** `w=[1,1], b=-0.5` 는 사실 OR 게이트라 XOR 4개 중 3개만 맞고 $(1,1)$ 에서 틀립니다. **어떤** $w,b$ 를 넣어도 최대 3개 — 대각선으로 놓인 1들을 직선 하나로 못 가르기 때문입니다. 이게 "선형 분리 불가능" 의 실체입니다.
3. **(3) 왜 '그냥' 쌓으면 안 되나.** $W_2(W_1x)$ 와 $(W_2W_1)x$ 가 **정확히 같은 값** `[26.0, 6.5]` 임을 확인했습니다. 활성함수가 없으면 2층이 1층으로 붕괴 — 깊이가 공짜로 표현력을 주지 않습니다. **비선형이 있어야 깊이가 의미** 를 얻습니다.
4. **(4) 비선형이 벽을 넘는다.** ReLU 은닉 뉴런 2개로 XOR 4개를 **모두** 맞혔습니다. 비결은 은닉 뉴런 $h_2=\text{ReLU}(x_1+x_2-1)$ 가 "둘 다 1" 일 때만 켜지고, 출력에서 $-2$ 로 눌러 $(1,1)$ 을 0 으로 되돌리는 것 — 두 직선을 "접어" 곡선 경계를 만든 셈입니다. **단, 오늘은 가중치를 우리가 손으로 넣었습니다.** 이걸 데이터가 스스로 찾게 하는 게 다음 편의 역전파입니다.

### 4.4 (실험) 직접 흔들어 보기

- **다른 활성함수**: `relu` 대신 시그모이드 `1/(1+np.exp(-z))` 로 은닉층을 바꿔 XOR 가 여전히 풀리는지 보세요(가중치는 다시 손으로 조정해야 함). "비선형이기만 하면" 된다는 걸 확인.
- **비선형 제거 실험**: (4)의 은닉층에서 `relu(...)` 를 그냥 `(W1 @ x + b1)` 로 바꿔 보세요. XOR 가 무너집니다 — 비선형이 없으면 (3)의 붕괴가 그대로 재현됩니다.
- **은닉 뉴런 수**: 은닉 뉴런을 2개→3개로 늘리고 가중치를 임의로 채운 뒤 출력이 어떻게 달라지는지 관찰. (아직 학습은 없지만) 표현할 수 있는 경계가 얼마나 유연해지는지 감을 잡습니다.
- **[[Day-014]] 와 잇기**: 로지스틱 회귀($w^\top x+b$ + 시그모이드)가 **은닉층 없는 1뉴런 신경망** 임을 코드로 대응시켜 보세요. "우리는 이미 Day-014 에서 신경망을 학습시켰다" 는 사실이 눈에 들어옵니다.

## 6. 생각거리 (Reflection)

> [!question] 스스로 답해 보기
> 1. **이미 만든 신경망.** [[Day-014]] 의 로지스틱 회귀($w^\top x+b$ 뒤 시그모이드)는 "은닉층 없는 1뉴런 신경망" 이라고 볼 수 있습니다. 그렇다면 로지스틱 회귀가 XOR 를 못 푸는 이유를 오늘 배운 "선형 분리 가능성" 으로 설명해 보세요.
> 2. **비선형이 왜 '반드시' 인가.** 실습 (3)에서 $W_2(W_1x)=(W_2W_1)x$ 였습니다. 이 등식이 "활성함수 없이 층을 100개 쌓아도 소용없다" 는 주장을 어떻게 증명하는지, 행렬 곱의 결합법칙을 근거로 한 문장으로 정리해 보세요.
> 3. **다음으로 가는 다리.** 오늘 XOR 를 푸는 가중치 $W_1,b_1,W_2,b_2$ 를 우리는 *손으로* 넣었습니다. 실제 문제(예: [[Day-025]] 의 `car`≈`automobile` 의미)에서는 이 가중치가 수백만 개라 손으로 못 정합니다. 그렇다면 "데이터를 보고 가중치를 자동으로 조정" 하려면 무엇이 필요할지 — 오차를 어떻게 각 가중치에 되돌려 줄지 — 를 [[Day-027]](역전파) 로 이어 상상해 보세요.

## 7. 오늘의 한 줄 요약 (Takeaway)

> [!summary]
> **신경망의 최소 부품인 뉴런 하나는 곧 선형 모델 $w^\top x+b$ 에 활성함수 $\phi$ 를 씌운 것** — [[Day-004]]·[[Day-014]] 에서 이미 다룬 그 식이다. 퍼셉트론(뉴런 1개)은 결정 경계가 직선 하나뿐이라 **선형 분리 가능한** AND·OR 는 풀지만 **XOR 는 못 푼다**(1들이 대각선으로 놓여 직선으로 못 가름). 층을 쌓되 **비선형 활성함수 없이** 쌓으면 $W_2(W_1x)=(W_2W_1)x$ 로 **선형 1층으로 붕괴** 해 표현력이 늘지 않는다 — 그래서 **비선형성이 딥러닝의 심장**이다. 층 사이에 ReLU 를 끼운 **2층 MLP** 는 입력 공간을 "접어" XOR 를 정확히 푼다(오늘 순전파로 손수 확인). 다만 오늘은 가중치를 손으로 넣었고, 이를 **데이터로 자동 학습** 하는 법 = **역전파(backpropagation)** 가 다음 편([[Day-027]])이다. 🛠️ 이 흐름은 Phase 3 마무리의 **B3 빌드(NumPy MLP → 미니 Transformer)** 로 이어진다.
